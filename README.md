BitesUAE is a fictional food delivery platform operating across 4 UAE cities (Dubai, Abu Dhabi, Sharjah, Ajman). This dashboard provides real-time insights into:

- **Order Performance** - Revenue, order volume, and trends
- **Delivery Operations** - On-time delivery rates, delays, and rider performance
- **Customer Analytics** - Customer segmentation, retention, and behavior
- **Restaurant Insights** - Top performers, cuisine analysis, and ratings
